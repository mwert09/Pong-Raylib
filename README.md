# Pong-Raylib
  Pong game with raylib library

# How to run?
  1. Unzip files.
  2. Open Raylib.sln file.
  3. Click Build and then Build Solution
  4. Ctrl + F5
  5. If you want to run from debug folder copy raylib.dll from project folder to Debug folder.
  6. Run Raylib.exe
  7. DONE!
